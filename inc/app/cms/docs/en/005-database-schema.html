Database Schema

The default Sitellite CMS includes the following database tables:

sitellite_bookmark

A list of bookmarks for each user.

sitellite_cache_file_list

Keeps a list of the cache files. Used by the saf.File.Store package, which is used by the saf.Cache package.

sitellite_category

A global category list, for cross-referencing data in multiple collections.

sitellite_filesystem

The file info table for the Web Files collection.

sitellite_filesystem_sv

The version history table for the Web Files collection.

sitellite_keyword

The global keyword list.

sitellite_lock

Keeps a list of locked items. Used by the cms.Workflow.Lock package.

sitellite_log

An internal activity log.

sitellite_message

The table for Sitellite's internal messaging system.

sitellite_mime_type

A list of known mime types.

sitellite_msg_attachment

The attachment list for the internal messaging system.

sitellite_msg_category

The category (ie. folders) list for the internal messaging system.

sitellite_msg_forward

The forward list for the internal messaging system.

sitellite_msg_queue

The sending queue for the internal messaging system.

sitellite_msg_recipient

The recipient list for the internal messaging system.

sitellite_news

The table for the News Stories collection.

sitellite_news_sv

The version history table for the News Stories collection.

sitellite_news_author

The author list for the News Stories collection.

sitellite_news_category

The category list for the News Stories collection.

sitellite_news_comment

The comment list for the News Stories collection.

sitellite_page

The table for the Web Pages collection.

sitellite_page_sv

The version history table for the Web Pages collection.

sitellite_prefs

The internal preference value table.

sitellite_property_set

A default table for the saf.Database.PropertySet collection.

sitellite_sidebar

The table for the Sidebars collection.

sitellite_sidebar_sv

The version history for the Sidebars collection.

sitellite_sidebar_position

A list of sidebar positions for the Sidebars collection.

sitellite_undo_sv

An undo history table.

sitellite_user

The default user collection table.
