Getting Started

Logging into Sitellite

To log into Sitellite, go to the following URL:

http://www.yourWebSite.com/sitellite

Then enter your username and password.

If this is a new installation and it is the first time you are logging in, you can enter with the username "admin" and password "admin" as well, then it is strongly recommended that you proceed to either change the "admin" password, or create a new master user and delete the "admin" account.
