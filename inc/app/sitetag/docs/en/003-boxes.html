Boxes

Here is the list of available boxes and their uses.

cloud

This box displays the list of used tags. The size of each tag is relative to its frequency of use.

Parameters

  • set: the name of the tag set we want to use.
  • showEdit: whether to show of not the edit tags link.

list-tags

This box displays the list of tags associated with the current page.

Parameters

  • set: the name of the tag set we want to use.
  • url: the url identifying the current page. It not specified, the page id will be used.
  • showEdit: whether to show of not the edit tags link.

list-items

This box displays the list of items (pages) pointed by the specified tag. This is the page called when you click on tags in list-tags of in cloud.

Parameters

  • set: the name of the tag set we want to use.
  • tag; the tag.

add

This box displays the edit tags form. It allows to add or remove tags from the current page, and to edit current page informations.

Parameters

  • set: the name of the tag set we want to use.
  • url: the url that identify the current page. If not specified, the current page id will be used.

related

This box displays a list of related pages, based on tags shared between pages.

Parameters

  • set: the name of the tag set we want to use.
  • num: the number of links to show.
